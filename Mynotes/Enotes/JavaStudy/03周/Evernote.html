  • 内部类
    • 成员内部类
      • 和外部类的属性同级
      • 内部类可以访问外部类所有的属性和方法
  • 静态内部类
    • 静态内部类只能调用外部类的静态属性
    • 为什么？
    • 创建静态内部类的对象，不会创建外部类的对象，外部类就不会在内存开辟空间来存放成员变量
    • 创建静态内部类对象，先在内存中加载外部类的字节码文件，会扫描外部类的所有静态属性，并存放在静态常量区，
    • 那么内部类对象就可以调用了
  • 接口内部类
    • 与静态内部类一样
    • 接口只有静态常量和抽象方法，接口属性默认添加public static final,而接口内部类相当于接口的属性
  • 局部内部类
    • 写在方法里
  • 匿名内部类
    • 抽象方法实现
    • 接口实现
    • 接口回掉，eg计算代码运行时间
//成员内部类
package test01;
public class Outter {
     //外部类属性
     String info = "外部类的info";
     private String str1 = "1";
     String str2 = "2";
     protected String str3 = "3";
     public String str4 = "4";
     final String str5 = "5";
     
     //成员内部类
     public class Inner{
          String info = "内部类属性";
          public int add(int a,int b) {
               return a+b;
          }
          public void print() {
               //调用外部类属性
               System.out.println(str1);
               System.out.println(str2);
               System.out.println(str3);
               System.out.println(str4);
               System.out.println(str5);
               //同名属性调用
               System.out.println(info);
               System.out.println(Outter.this.info);
          }
          
     }
     //外部类方法
     public int add(int a,int b) {
          Inner inner = new Inner();
          return inner.add(a, b);
     }
}
